# Motion Graphics

Generate motion graphics videos from plain text descriptions — no editing skills needed.

## Quick Start

```
Create a motion graphic: animated bar chart showing monthly revenue
growing from $10K to $100K with spring animations and dollar labels
```

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `prompt` | Yes | Describe the animation you want |
| `duration_frames` | No | Length in frames (default: 150 = 5sec at 30fps) |
| `fps` | No | Frames per second (default: 30) |
| `width` | No | Width in pixels (default: 1920) |
| `height` | No | Height in pixels (default: 1080) |

## Example Usage

**Text animation:**
```
Make a typewriter animation that types "WELCOME TO THE FUTURE"
then each word highlights in gold one at a time
```

**Data dashboard:**
```
An animated dashboard with 3 metric cards (Users: 50K, Revenue: $2M,
Growth: 340%) — numbers count up with spring physics
```

**Social media promo:**
```
A vertical Instagram Reel intro with bold text "5 AI TOOLS YOU NEED"
that slams in word by word with screen shake
```

## Setup

1. Set `OPENAI_API_KEY` in environment
2. Template auto-installs on first run

## How It Works

1. AI analyzes your prompt and detects needed capabilities
2. Generates a React/Remotion animation component
3. Auto-corrects any compilation errors (up to 3 retries)
4. Renders to MP4 with Remotion

## What It Can Make

- Kinetic typography & text animations
- Data visualizations (charts, graphs, counters)
- Chat/messaging UI mockups
- Logo reveals & brand intros
- Abstract motion & geometric animations
- 3D scenes (via Three.js)
- Social media templates

## Limitations

- Requires OpenAI API key
- Complex 3D scenes may need manual refinement
- Generated code quality depends on prompt specificity
- Landscape (1920x1080) by default
