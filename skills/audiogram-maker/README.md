# Audiogram Maker

Turn audio clips into visual videos with waveforms and synced captions.

## Quick Start

```
Turn my podcast clip at ~/audio/episode.wav into an audiogram
titled "Why AI Agents Will Replace SaaS"
```

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `audio_path` | Yes | Path to audio file |
| `title` | Yes | Title displayed on audiogram |
| `cover_image` | No | Cover art image path |
| `visualizer` | No | `oscilloscope` or `spectrum` (default: oscilloscope) |
| `color` | No | Visualizer color hex |

## Example Usage

**Podcast clip:**
```
Create an audiogram from my-interview.wav with the title
"Building in Public" and a spectrum visualizer in blue
```

**Voice note:**
```
Make a video from voice-memo.wav titled "Quick Update"
with my logo at cover.png
```

## Setup

- ffmpeg must be installed
- Whisper.cpp auto-downloads on first run
- No API keys required — fully local

## How It Works

1. Copies audio + cover art into Remotion project
2. Runs Whisper.cpp for word-level transcription
3. Calculates video duration from audio length
4. Renders with waveform visualization + synced captions
5. Outputs square MP4 (1080x1080)

## Limitations

- Square format (1080x1080) by default
- Long files (>30min) may be slow to render
- Use .wav for large files to avoid memory issues
- English optimized by default
