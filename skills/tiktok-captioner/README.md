# TikTok Captioner

Add TikTok-style animated word-by-word captions to any video.

## Quick Start

```
Add captions to my video at ~/videos/clip.mp4
```

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `video_path` | Yes | Path to video file |
| `highlight_color` | No | Active word color (default: #39E508 lime green) |
| `caption_switch_ms` | No | Caption refresh rate in ms (default: 1200) |
| `language` | No | Transcription language (default: en) |

## Example Usage

**Basic captioning:**
```
Caption this video: /path/to/my-video.mp4
```

**Custom style:**
```
Add captions to my-video.mp4 with a red highlight color
and show one word at a time
```

## Setup

- ffmpeg must be installed
- Whisper.cpp model auto-downloads on first run (~1.5GB)
- No API keys required — runs fully local

## How It Works

1. Copies your video into the Remotion project
2. Extracts audio and runs Whisper.cpp speech-to-text
3. Gets word-level timestamps for precise sync
4. Renders captions with spring animations and word highlighting
5. Outputs final MP4 with captions baked in

## Limitations

- First run downloads ~1.5GB Whisper model
- Processing time scales with video length (~1x realtime)
- English optimized by default (switch model for other languages)
- Vertical (1080x1920) output by default
