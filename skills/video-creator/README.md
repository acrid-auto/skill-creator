# Video Creator

Generate short-form AI videos from a text prompt — script, visuals, voiceover, and captions all automated.

## Quick Start

```
Make me a video about "The Rise of AI Agents" covering how autonomous
agents are changing software development
```

## Parameters

| Parameter | Required | Description |
|-----------|----------|-------------|
| `title` | Yes | Video title (shown in intro) |
| `topic` | Yes | What the video covers |
| `voice_id` | No | ElevenLabs voice ID (default: Rachel) |
| `output_path` | No | Output file path |

## Example Usage

**Educational explainer:**
```
Create a video titled "How Bitcoin Works" explaining the basics of
blockchain technology for beginners
```

**Storytelling:**
```
Generate a video called "The Last Library" — a short story about the
last physical library in a fully digital world
```

## Setup

1. Set environment variables:
   ```
   OPENAI_API_KEY=sk-...
   ELEVENLABS_API_KEY=...
   ```
2. Template auto-installs on first run

## How It Works

1. GPT-4 writes an 8-10 sentence story from your topic
2. GPT-4 generates image descriptions for each scene
3. DALL-E 3 creates visuals (1024x1792 vertical)
4. ElevenLabs narrates with word-level timing
5. Remotion assembles with zoom animations + blur transitions
6. Renders to MP4 (1080x1920, 30fps)

## Limitations

- Requires OpenAI and ElevenLabs API keys (paid APIs)
- Generation takes 2-5 minutes depending on story length
- Vertical format only (TikTok/Reels/Shorts)
- English language only by default
